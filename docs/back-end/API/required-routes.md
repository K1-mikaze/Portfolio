# Required Routes

All of this information will be gotten from the database

## Homepage

The homepage need two routes:

### Left Sidebar

**Route**: This route need to return the following information

- GitHub link
- LinkedIn link
- Proton Mail link
- WhatsApp Mail link

**Link to other Pages of the Webside**

- Portfolio
- About me

### Blog Pages & Right Sidebar

**Route**: This route need to return the following information:

- **Card**: This card should be in the center of the screen and on click will move you to the blog it represents, the card will have the
  card title, tags of the technologies used in it, A photo of the information or something that represent the blog.

- **Right Sidebar**: The Sidebar should contain a Search input field at top, and in the rest of the sidebar should contain all the tags
  used between all the blogs, on click on those tags, the searcher will filter among all the pages and show the pages that use those tags
